
# Operation Theatre Management System (OTMS)

## Objective
Manage surgical workflows and OT operations.

## Scope
### Scheduling
- OT calendar
- Surgeon allocation

### Pre-Operative
- Checklist
- Patient readiness

### Surgery
- Procedure details
- Staff allocation

### Post-Operative
- Recovery tracking
- Notes

### Billing
- Procedure charges

## Integration
- Input: Admission
- Output: Billing + clinical notes

## Detachability
- Can run as standalone OT system
